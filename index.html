VDiscover makes large-scale vulnerability discovery possible
using state-of-the-art Machine Learning techniques.










Why

With sustained growth of software complexity, finding security vulnerabilities in operating systems has become an important necessity. Unfortunately, the use of costly techniques on a large amount of testcases to analyze is still an issue today. The lack of source code access restricts even more the possibilities to find vulnerabilities at large scale.

We were motivated by these reasons to design and develop a predictive approach to vulnerability discovery.

How

Given a costly vulnerability discovery procedure and a large amount of testcases to analyze,







VDiscover is trained to predict the result of a costly analysis extracting different sets of features using a fully automatic and very lightweight approach. Features are directly from executable files without the source code.







Later, our tool can predict the outcome of the vulnerability discover procedure previously used.







For further details, a technical report is available as well as an open-source prototype.

Where

VDiscover is open-source, you can test it here:

git clone https://github.com/CIFASIS/VDiscover.git
cd VDiscover
python setup.py install --user
